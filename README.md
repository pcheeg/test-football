# Goal Guess — Test Build

A five-round Premier League goals guessing prototype using the variable-speed football slider.

## Test players

1. Alan Shearer — 260
2. Steven Gerrard — 120
3. Riyad Mahrez — 82
4. Willian — 47
5. Benjamin Mendy — 2

## GitHub Pages

Upload every file in this folder to the root of a GitHub repository. In GitHub, open **Settings → Pages**, select **Deploy from a branch**, then choose **main** and **/root**.

The service worker makes the app available offline after its first successful load from GitHub Pages.
